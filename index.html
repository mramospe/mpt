Metaprogramming Tools for C++
MPT Documentation

The Metaprogramming Tools (MPT) package is a header-only C++ library which offers general utilities to handle compile-time expressions. Each header in the package corresponds to a non-independent library:

  • arfunctors: arithmetic and relational functors
  • keywords: objects configurable by keywords
  • members: wrappers and checkers for class members
  • signature: handle (member) function and functor signatures
  • smart_enum: smart enumeration types
  • soa: containers with struct-of-arrays memory layout
  • typed_any: typed any objects
  • types: handle types in template parameter lists
  • values: handle values in template argument lists

The package is built on top of the standard library using the C++20 features.
