37  /*\brief Represent an object that can hold any of a set of types, saving the
38  index of the type
39 
40  The behaviour is quite similar to std::variant, but in this case the size of
41  the object is not determined by the size of the biggest object from the
42  template argument list. The value is stored on a std::any object and the
43  type index is saved as an integer. The actual value can be accessed using
44  mpt::visit_typed_any (similarly to std::variant).
45 
46  This class is meant to be used when it is needed to have an object that
47  might be any of a set of types that have very different size in memory.
48 
49  It is not recommended to use this class on lightweight classes stored in
50  large containers due to the cost of having an additional integer saving the
51  type of each of them. If all of them are of the same type, it is probably
52  better to use std::variant since all the containers store a pointer and an
53  integer (i.e. are of the same size in memory).
54  */
