Detailed Description

This header defines tools to create and handle smart enumeration types. A smart enumeration type is an enumeration type that can be created/converted from/to a string of characters. The definition of smart enumeration types is done through the MPT_SMART_ENUM macro. Consider, for example:

MPT_SMART_ENUM(position, position_properties, int, unknown_position, bottom,
middle, top);
auto some_function(position p) {
switch (p) {
case (unknown_position):
return ...;
case (bottom):
return ...;
case (middle):
return ...;
case (top):
return ...;
}
}

In this case we are creating an enumeration type called position, whose properties will be saved in a new type called position_properties. The enumeration type will be of type int, whenever there is a problem with the processing of the enumeration type (like for conversion to a string of characters) the value unknown_position will be returned, and bottom, middle and top are the members of the enumeration type, with values 1, 2 and 3.

Declare a smart enumeration type.

The arguments are:

  • enum_name: name of the enumeration type, exposed in the current scope.
  • enum_properties_name: name of the structured object containing the properties of the enumeration type.
  • type: underlying type of the enumeration type.
  • unknown: name of the value that is reserved in case of problems parsing enumeration types.
  • ...: list of names that define the members of the enumeration type.

The exposed enum_properties_name type will contain the following members

  • underlying_type: type provided in type
  • va_args_with_unknown: (for internal use only) representation of the input arguments as a string, enclosed in a dedicated type
  • va_args: (for internal use only) similar to va_args_with_unknown, but without the unknown value included
  • unknown_value: value that is used in case of problems with parsing enumeration values
  • size: number of names in the enumeration type, wihtout the unknown
  • size_with_unknown: total number of names in the enumeration type
  • names: names of the enumeration values, saved as std::string_view objects
  • names_with_unknown: similar to names but with the unknown value included
  • values: values of the enumeration type saved in an array
  • values_with_unknown: similar to values but with the unknown value included

Invoking this macro also creates a function called properties that is used in order to access the aforementioned class via mpt::smart_enum_properties and mpt::smart_enum_properties_t.
