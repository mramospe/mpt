Detailed Description

template<class Container, class... Fields>
class mpt::soa_proxy< Container, mpt::types< Fields... > >

Proxy for an element of a container.

A proxy acts as a mpt::soa_value object, but any modification (including assignment) will modify the associated contents in the referenced container. The construction of this object is limited to copy and move constructors (where the assignment of values is not done).
